# Blog Website

This is a simple blog website built with HTML, CSS, and JavaScript. It displays a list of blogs on the index page. When a blog is clicked, it redirects to a detail page where the full content of the blog is displayed.

## Project Structure

- `index.html`: This is the main page of the website. It displays a list of featured blogs.
- `blogs.html`: This is the page where all blogs are listed.
- `blog-details.html`: This page displays the full content of a blog. The blog to display is determined by the `blog` parameter in the URL.
- `about.html`: This is the page with information about the author.
- `style.css`: File containing the CSS properties of the project.
- `app.js`: File containing the JS codes of the project.
- `blogs-image/`: This directory contains the images for the blogs.
- `img/`: This directory contains the images for project.

## How to Run

To run this website, simply open `index.html` in your web browser.
